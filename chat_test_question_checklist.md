# Course 4186 Chat Test Question Checklist

This file is a broad test-question checklist for the Chat part only.
It is not a bug report yet.
It is the question bank I would use to systematically test the chat experience from a student perspective.

## 16. What This Checklist Is Meant to Catch

- Robotic or repetitive answers
- Lost context in follow-up turns
- Multi-concept questions answered only partially
- Code requests returning no code block
- Code-only requests returning too much prose
- Course citations missing when they should exist
- Wrong citations or mismatched references
- Out-of-scope questions not being clearly refused
- JSON / placeholder / transport leakage into student-facing output
- Markdown rendering that looks broken or not product-grade
